Comments

Comments are some text or code written in your code to give an explanation about the code, and not visible to the user. Comments which are used for HTML file are known as HTML comments. Anything written between these tags will be ignored by the browser, so comments will not be visible on the webpage. Comments of any code make code easy to understand and increase readability of code. Comments are also part of the code, which gives an explanation of the code. How to add comment In HTML You can add comments in your HTML file using tag. So if you will write anything between theses comment tag that will be treated as comment and browser will not read it.
